## API
|接口名称|请求方式|功能描述|
|---|---|---|
|**creditScore**|POST|获取数据风险等级和标签|
|**delBWList**|DELETE|删除黑/白名单|
|**describeBWList**|GET|获取黑/白名单列表|
|**setBWList**|POST|设置黑/白名单|
