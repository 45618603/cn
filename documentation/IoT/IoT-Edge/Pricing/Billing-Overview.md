# 计费概述

## 收费项目

IoT Edge按照Edge系统部署用量收费，计费项包括：已部署Edge系统的终端设备数

## 地域与可用区

IoT Edge目前在各个地域的各个可用区中价格一致。

## 欠费/到期说明

### 包年包月计费实例

到期后，计费状态会被标记为到期状态，Edge与物联网中心消息收发服务，数据收发服务，系统更新，模块更新，设备管理等服务不可用。如7天内未续费，则在7天后在物联网中心的Edge信息会被删除，同时该Edge的备份文件也会被删除，数据不能恢复。
